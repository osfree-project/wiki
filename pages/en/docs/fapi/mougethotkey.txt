====== MouSetHotKey ======

Returns mouse button or buttons defined as system hot key.

===== Syntax =====
<code c>
 MouSetHotKey(HotKey, DeviceHandle)
</code>

===== Parameters =====


  * HotKey ()
  * DeviceHandle (HMOU)

===== Return Code =====

  * ERROR_MOUSE_NO_DEVICE
