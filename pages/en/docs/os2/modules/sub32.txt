====== SUB32 ======

32-bit implementation of Vio/Mou/Mon/Kbd functions compatible with EMXWRAP 32-bit interface. Mapped to virtual DLL [[EMXWRAP]]. Don't call this functions directly, use EMXWRAP.DLL. For IBM 32-bit interface use [[OS2CHAR2]].DLL
