====== BASIC ======

===== Description =====

BASIC is a Disk BASIC dialect, compatible with IBM Personal Computer BASIC.

===== Keywords =====

List of IBM Personal Computer BASIC 1.10 words

  * Dsk - Disk BASIC (BASIC.COM)
  * Adv - Advanced BASIC (BASICA.COM)

  * Gw - GW-BASIC 3.23
