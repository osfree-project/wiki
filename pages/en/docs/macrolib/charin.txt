====== @CharIn ======

===== Brief =====

Read char from keyboard

===== BIOS Interrupt =====

[[en:docs:bios:api:int16|INT16H]]: [[en:docs:bios:api:int16:00|00H]]

===== Family API =====

[[en:docs:fapi:]]

===== Syntax =====

<code asm>
INCLUDE BIOS.INC

@CharIn
</code>

===== Parameters =====

none

===== Return =====

  * AH = BIOS scan code
  * AL = ASCII character
