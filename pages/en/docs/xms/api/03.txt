====== AH=03H ======

===== Version =====

2 and higher

===== Brief =====

Global Enable A20

===== Family API =====

none

===== Input =====

  AH = 03h

===== Return =====

  * AX = 0001h if the A20 line is enabled, 0000h otherwise
  * BL = Error code
