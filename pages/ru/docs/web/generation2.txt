===== Инфраструктура 2 поколения =====

**Замечание**: 2-е поколение сайта osfree.org сейчас уже в прошлом, но старые ссылки все еще работают.

osFree.org второго поколения состоит из файла index.php, которому передаются доп. аргументы в PHP-скрипт.

Для различного контента поддерживались следующие аргументы:

<code> ''добавить сюда'' </code>

Кроме того, osfree.org 2-го поколения содержит файлы для перенаправления в новое место расположения контента.

|index.html          |-> index.php                             |Бывшая исходная точка сайта|
|top.html             |-> index.php                             |Бывший фрейм с лого|
|menu.html         |-> index.php                              |Бывший фрейм с меню|
|start.html          |-> index.php                              |Бывшая страница новостей|
|objective.html    |-> index.php                              |Бывшая страница с описанием проекта|
|developing.html |-> index.php?gobaby=coding    |Бывшая страница с инструкциями для разработчиков|
|licence.html       |-> index.php?gobaby=license   |Бывшая страница о лицензиях (замечание: имя страницы содержало ошибку)|
|members.html    |-> index.php?gobaby=develop  |Бывший список членов проекта|
|download.html    |-> index.php?gobaby=download  |Бывшая секция скачивания бинарников|
|contact.html       |-> index.php?gobaby=forum     |Бывшая страница с контактной информацией|
