===== Инфраструктура 3 поколения =====

Также, osfree.org 3-го поколения содержит такие файлы для перенаправления со старых URL-ов в текущее место.

<code>  Ссылки на файлы сайта 1-го поколения. Перенаправляются сейчас на.... </code>

|index.html        |-> index.php                                |Бывшая стартовая точка сайта|
|top.html           |-> index.php                               |Бывший фрейм с лого|
|menu.html        |-> index.php                                |Бывший фрейм с меню|
|start.html         |-> index.php                                |Бывшая страница новостей проекта|
|objective.html   |-> index.php                                |Бывшая страница с описанием проекта|
|developing.html   |-> index.php?gobaby=coding |Бывшая страница с инструкциями для разработчиков|
|licence.html      |-> index.php?gobaby=license     |Бывшая страница про лицензии (замечание: имя файла содержало ошибку)|
|members.html  |-> index.php?gobaby=develop     |Бывший список членов проекта|
|download.html  |-> index.php?gobaby=download  |Бывшая страница для скачивания бинарников|
|contact.html     |-> index.php?gobaby=forum        |Бывшая страница с контактной информацией|

^(ссылки на файлы сайта 2-го поколения)                            ^
|index.php?gobaby=...      |-> doku/en:...|
