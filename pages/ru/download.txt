===== Последние файлы, доступные для скачивания =====

Основная часть файлов osFree в данный момент доступна для скачивания через GitHub. В текущее время проект osFree находится в глубокой альфа-стадии. Для тестиования доступен ряд файлов в виде инсталляционных пакетов, кроме того, для просмотра доступны исходники через WebGit. Кроме того, есть прямой анонимный доступ к Git: [[https://github.com/osfree-project/osfree/|наш репозиторий]], используя ваш любимый Git-клиент. Более подробно о доступе к информации через Git вы можете прочитать на сайте [[https://github.com|GitHub]]. Тем не менее, основная часть этой информации есть на этом сайте.

Также, доступен наш [[en:demo|Демо CD]].
